## 📚 Jorenoe 文档

### 🚀 概览

**Jorenoe** 是一款高效的 Net 封装快速开发工具包，支持多种数据库及高性能数据访问方式。

#### 📌 支持特性

* **数据库支持**：MySQL、SQL Server
* **ORM 框架**：Dapper、Entity Framework Core
* **消息队列**：RabbitMQ 集成
* **缓存**：Redis 高效数据存储 等
